Core API#

Detect model collapse for PyTorch module

collapsedetect.is_collapse(*tensors: torch.Tensor, eps: float = 1e-05) → bool#

Detect if the given tensors are collapsed. Requires more than one 1D tensor, or 2D tensors with the first dimension greater than 1.

Parameters

  • tensors – tensors to be detected. Can be either 1D or 2D. All tensors are unsquashed to 2D (if needed) and are concatenated to a tensor of [N, D], which N > 1.

  • eps – The result max energy will be compared with this value.

Returns

whether input tensors are collapsed.

>>> is_collapse(torch.ones((2, 3), dtype=torch.float32))
True
>>> is_collapse(torch.randn((3,)), torch.randn((3,)))
False
